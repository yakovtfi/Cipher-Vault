"# Cipher-Vault" 
יעקב תפילין חרמון 314958950

בניתי פה שרת שבא אני יוצר משתמש ואז המשתמש יכול להצפין הודעה ואז גם להפוך את זה להודעה רגילה בחזרה

השתמשנו פה ב mongodb בשביל המשתמשים והשתמשנו בsupabase בשביל ההודעות

ברגע שיצרנו משתמש  נוכל אחר ליצור הודעה על ידי הכנסת שם המשתמש והסיסמה ורק אז נוכל לשלוח הודעה

יש לנו שלושה ראוטים ועוד ראוט index שמכיל בתוכו את כל השלושה וממנו אני שולח לapp

כדי להריץ נתקין את כל הספריות על ידי: npm install ואז נריץ: node app.js
